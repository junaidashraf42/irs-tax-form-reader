# IRS Tax Form Reader

Open-source AI/ML tool for reading and extracting data from scanned IRS tax forms.

## Overview

This project provides a local, CPU-friendly solution for extracting structured data from PDF tax forms using dual OCR engines (Tesseract + PaddleOCR) and LayoutLM for document understanding.

## Supported Forms

- Form 1040, 1065, 1120, 1120S
- Schedule 1, C, E1, E2, K, L, M1, M2, 8825-1, 8825-2
